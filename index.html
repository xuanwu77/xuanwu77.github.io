CSS hack

发表于 2017-07-04

什么是CSS hack？

不同的浏览器对某些CSS代码解析会存在一定的差异，因此会导致不同浏览器下给用户展示的页面效果不一样，碰到这样的情况就需要开发人员通过针对不同的浏览器对应写出不同的CSS代码，这种技术有个专门的名称就是CSS Hack。

搭个自己的窝啦

发表于 2017-05-18

今天完成了HEXO+Github搭建自己的博客，当然，网上一堆文章，很好找，但也杂。这里推荐这一篇文章，2017年最新基于hexo搭建个人免费博客——从零开始。文章从HEXO安装，域名绑定，还有编写文章并部署到Github上以及安装主题等都有详细说明。半个月前，我确实是这么做了，不过呢，出现一个问题，我的博客在本地服务器可以看到，却无法部署到Github 。最后解决的方法是根目录配置文件_config.yml中拉到底部的部署配置有错，应该是与hexo版本有关，于是修改成：

deploy:
  type: git
  repo: https://github.com/xuanwu77/xuanwu77.github.io.git
  branch: master

结果是成功了！那以后有可能的话，不犯懒的话，应该会更新博客吧。
